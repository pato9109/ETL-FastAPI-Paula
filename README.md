Este proyecto demuestra una solución completa de Ingeniería de Datos, integrando la extracción de datos, transformación con Pandas y una capa de servicio (API) protegida con seguridad OAuth2.

Tecnologías Utilizadas
Lenguaje: Python.

Framework API: FastAPI.

Procesamiento de Datos: Pandas.

Base de Datos: SQLite / SQLAlchemy.

Servidor: Uvicorn.
